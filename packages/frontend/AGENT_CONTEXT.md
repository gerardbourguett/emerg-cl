# Frontend Context

## Architecture
- Framework: React (via Vite/React Router)
- Styling: TailwindCSS v4
- Map: Mapbox/Leaflet (To be implemented)

## Key Files
- `src/entry.client.tsx`: Client entry point.
- `src/routes.ts`: Route definitions.

## Common Tasks
- Add new route: Create component in `src/routes/` and add to route config.
- Styling: Use utility classes.
